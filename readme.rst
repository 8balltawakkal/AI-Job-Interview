谁使用PHP构建网站。其目标是使您能够开发项目
通过提供以下内容，比从头开始编写代码要快得多
丰富的库，用于执行常见任务，以及简单的
接口和逻辑结构来访问这些库。 CodeIgniter让
您可以通过减少所需的代码量来创造性地专注于项目
对于给定的任务。
