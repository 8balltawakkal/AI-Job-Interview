##################
Обработка естественного языка
###################

это область информатики, искусственного интеллекта, связанная с взаимодействием между компьютерами и человеческими (естественными) языками, и, в частности, связана с программированием компьютеров для плодотворной обработки больших данных естественного языка.

Проблемы при обработке естественного языка часто связаны с распознаванием речи, пониманием естественного языка и генерацией естественного языка.

*******************
CodeIgniter
*******************

CodeIgniter - это среда разработки приложений - набор инструментов - для людей
которые создают веб-сайты с использованием PHP. Его цель - дать вам возможность разрабатывать проекты
намного быстрее, чем если бы вы писали код с нуля, предоставляя
богатый набор библиотек для часто используемых задач, а также простой
интерфейс и логическая структура для доступа к этим библиотекам. CodeIgniter позволяет
вы творчески сосредотачиваетесь на своем проекте, сводя к минимуму количество необходимого кода
для данной задачи.
